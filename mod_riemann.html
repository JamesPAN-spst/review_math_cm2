Riemann 和

Sommes de Riemann et passage à l'intégrale

核心定义

公式 Riemann 和 → 定积分
\[\int_a^b f(x)\,dx = \lim_{n\to\infty}\frac{b-a}{n}\sum_{k=0}^{n-1}f\!\left(a+k\cdot\frac{b-a}{n}\right)\]

最常用的特殊形式（\(a=0,\,b=1\)）：

\[\lim_{n\to\infty}\frac{1}{n}\sum_{k=1}^{n}f\!\left(\frac{k}{n}\right)=\int_0^1 f(x)\,dx\]

识别三步法

方法 系统化识别 Riemann 和

第一步 · 提取 \(\Delta x\)：找到求和中的公因子 \(\frac{1}{n}\) 或 \(\frac{c}{n}\)，这就是 \(\Delta x = \frac{b-a}{n}\)。

第二步 · 还原 \(f(x_k)\)：将求和的通项写成 \(\frac{1}{n}\cdot g(k/n)\) 的形式，其中 \(g\) 就是 \(f\)。

第三步 · 确定积分区间：\(k\) 从 1 到 \(n\) → \([0,1]\)；从 1 到 \(2n\) → \([0,2]\)；从 \(n+1\) 到 \(2n\) → \([1,2]\)（需偏移变量）。

易错 积分区间不总是 \([0,1]\)

当求和上限为 \(kn\) 时，积分上限变为 \(k\)。当求和下限不从 1 开始（如从 \(n+1\) 开始），需先做指标平移。

逐题详解（TD Exercice 2）

1. \(S_n = n\left(\frac{1}{(n+1)^2}+\frac{1}{(n+2)^2}+\cdots+\frac{1}{(2n)^2}\right)\)

改写：\(S_n = \sum_{k=1}^{n}\frac{n}{(n+k)^2} = \frac{1}{n}\sum_{k=1}^{n}\frac{1}{(1+k/n)^2}\)

识别：\(\Delta x = 1/n\)，\(f(x) = \frac{1}{(1+x)^2}\)，区间 \([0,1]\)。

\(\lim S_n = \int_0^1\frac{dx}{(1+x)^2} = \left[-\frac{1}{1+x}\right]_0^1 = -\frac{1}{2}+1 = \boxed{\frac{1}{2}}\)

2. \(S_n = \frac{1}{n}\left(\sin\frac{\pi}{n}+\sin\frac{2\pi}{n}+\cdots+\sin\frac{n\pi}{n}\right)\)

\(S_n = \frac{1}{n}\sum_{k=1}^{n}\sin\frac{k\pi}{n}\)

识别：\(f(x)=\sin(\pi x)\)，区间 \([0,1]\)。

\(\lim S_n = \int_0^1\sin(\pi x)\,dx = \left[-\frac{\cos(\pi x)}{\pi}\right]_0^1 = -\frac{-1}{\pi}+\frac{1}{\pi} = \boxed{\frac{2}{\pi}}\)

3. \(S_n = \sum_{k=1}^{n}\frac{k}{k^2+n^2}\)

提因子：\(= \frac{1}{n}\sum_{k=1}^{n}\frac{k/n}{(k/n)^2+1}\)

识别：\(f(x)=\frac{x}{x^2+1}\)，区间 \([0,1]\)。

\(\lim S_n = \int_0^1\frac{x}{x^2+1}dx = \frac{1}{2}\left[\ln(x^2+1)\right]_0^1 = \boxed{\frac{\ln 2}{2}}\)

4. \(S_n = \frac{\sqrt{1}+\sqrt{2}+\cdots+\sqrt{n-1}}{n\sqrt{n}}\)

\(S_n = \frac{1}{n}\sum_{k=1}^{n-1}\frac{\sqrt{k}}{\sqrt{n}} = \frac{1}{n}\sum_{k=1}^{n-1}\sqrt{k/n}\)

识别：\(f(x)=\sqrt{x}\)，区间 \([0,1]\)。（上限写成 \(n-1\) 不影响极限）

\(\lim S_n = \int_0^1\sqrt{x}\,dx = \left[\frac{2}{3}x^{3/2}\right]_0^1 = \boxed{\frac{2}{3}}\)

5. \(S_n = \sqrt[n]{e^{1+(2/n)^2}\cdot e^{1+(3/n)^2}\cdots e^{1+((n-2)/n)^2}}\)

关键技巧：先取对数！

\(\ln S_n = \frac{1}{n}\sum_{k=2}^{n-2}\left(1+\frac{k^2}{n^2}\right) = \frac{1}{n}\sum_{k=2}^{n-2}\left(1+(k/n)^2\right)\)

识别：\(f(x) = 1+x^2\)，区间 \([0,1]\)。

\(\lim \ln S_n = \int_0^1(1+x^2)dx = 1+\frac{1}{3} = \frac{4}{3}\)

\(\lim S_n = \boxed{e^{4/3}}\)

6. \(S_n = \sum_{k=1}^{2n}\frac{k}{n^2+k^2}\)

\(= \frac{1}{n}\sum_{k=1}^{2n}\frac{k/n}{1+(k/n)^2}\)

注意上限是 \(2n\)！所以 \(k/n\) 从 \(1/n \to 0\) 到 \(2n/n = 2\)。

识别：\(f(x)=\frac{x}{1+x^2}\)，区间 \(\mathbf{[0,2]}\)。

\(\lim S_n = \int_0^2\frac{x}{1+x^2}dx = \frac{1}{2}\ln(1+x^2)\Big|_0^2 = \boxed{\frac{\ln 5}{2}}\)

拓展知识

拓展 调和级数的积分估计

TD Exercice 11 利用 Riemann 和的思想反向操作——用积分估计求和：

\[\int_{n+1}^{2n+1}\frac{dx}{x} \le \sum_{p=n+1}^{2n}\frac{1}{p} \le \int_n^{2n}\frac{dx}{x}\]

两边都等于 \(\ln 2\)（夹逼），因此 \(\lim_{n\to\infty}\sum_{p=n+1}^{2n}\frac{1}{p} = \ln 2\)。

更一般地：\(\lim_{n\to\infty}\sum_{p=n}^{kn}\frac{1}{p} = \ln k\)。

拓展 非标准 Riemann 和的处理

有时求和的形式不直接匹配标准模板，需要先做代数变形：

① 指标平移：\(\sum_{k=n+1}^{2n}\) → 令 \(j = k - n\)，变为 \(\sum_{j=1}^{n}\)，其中 \(k/n = 1 + j/n\)。

② 取对数：乘积形式 \(\prod\) → 对数后变为 \(\sum\)（如例题 5）。

③ 分母提 \(n^2\)：形如 \(\frac{k}{n^2+k^2} = \frac{1}{n}\cdot\frac{k/n}{1+(k/n)^2}\)。
