反常积分

Intégrales impropres — 敛散性判定全攻略

定义与分类

定义 两类反常积分

无穷积分（积分上限为 \(\infty\)）：\(\displaystyle\int_a^{+\infty}f(t)\,dt = \lim_{A\to+\infty}\int_a^A f(t)\,dt\)

瑕积分（被积函数在端点处无界）：\(\displaystyle\int_a^b f(t)\,dt = \lim_{\varepsilon\to 0^+}\int_{a+\varepsilon}^b f(t)\,dt\)（瑕点为 \(a\)）

p-判别法 — 最核心的工具

公式 p-判别法：两套准则，方向相反！
\[\int_1^{+\infty}\frac{dt}{t^p}:\quad \begin{cases}\text{收敛} & p > 1 \\ \text{发散} & p \le 1\end{cases} \qquad\qquad \int_0^{1}\frac{dt}{t^p}:\quad \begin{cases}\text{收敛} & p < 1 \\ \text{发散} & p \ge 1\end{cases}\]
⚠ 最高频错误 条件反转

无穷端要 \(p>1\)，瑕点端要 \(p<1\)。记忆口诀：「远严近宽」。

对比表格：

收敛条件发散条件临界 \(p=1\)
\(\int_1^{+\infty}1/t^p\)\(p > 1\)\(p \le 1\)发散（\(\ln t \to \infty\)）
\(\int_0^1 1/t^p\)\(p < 1\)\(p \ge 1\)发散（\(\ln t \to -\infty\)）

比较判别法

公式 直接比较（放缩法）

设 \(0 \le f(x) \le g(x)\)：

\[\int g \text{ 收敛} \implies \int f \text{ 收敛}, \qquad \int f \text{ 发散} \implies \int g \text{ 发散}\]
公式 极限比较判别法
\[\text{设 } f,g > 0,\quad L = \lim \frac{f(x)}{g(x)} \implies \begin{cases} 0
⚠ 易错 \(L=0\) 或 \(L=\infty\) 时只能单向推导

\(L=0\) 说明 \(f\) 比 \(g\) 衰减更快，但「多快」未知。例：\(f = 1/(x\ln x)\)，\(g = 1/x\)，\(L=0\)，\(\int g\) 发散，但 \(\int f\) 也发散。必须换更精确的 \(g\)。

\(L=\infty\) 说明 \(f\) 比 \(g\) 衰减更慢，但 \(g\) 收敛不能排除 \(f\) 也收敛的可能。例：\(f=1/x^{1.5}\)，\(g=1/x^2\)，\(L=\infty\)，两者都收敛。

试探函数选取策略

方法 「抓大头」确定阶数

有理函数：分子最高次 / 分母最高次 = 整体阶数。例：\(\frac{3x^2+5}{x^5+2x^2+1} \sim \frac{3}{x^3}\)，选 \(g = 1/x^3\)。

根号：\(\frac{1}{\sqrt{x(x+1)(x+2)}} \sim \frac{1}{x^{3/2}}\)（根号内是 3 次）。

方法 指数必杀多项式

对任意实数 \(n\)：\(\displaystyle\lim_{x\to+\infty}\frac{x^n}{e^x} = 0\)。因此 \(\int x^n e^{-x}dx\) 永远收敛。

甚至 \(e^{-\sqrt{x}}\) 也足够杀死任何多项式：\(\int x^n e^{-\sqrt{x}}dx\) 收敛。

方法 对数太慢

\(\ln x\) 增长极慢，不影响敛散性的「阶」：\(\frac{\ln x}{x^p} \sim 1/x^{p-\varepsilon}\)（对任意 \(\varepsilon > 0\)）。

但 \(\frac{1}{\ln x}\) 衰减极慢，比 \(1/x\) 还慢 → \(\int_2^{+\infty}\frac{dx}{\ln x}\) 发散。

方法 有界量直接放缩

\(|\sin x| \le 1\)，\(|\cos x| \le 1\)，\(\arctan x \le \pi/2\)。直接替换为常数后看剩余的 \(p\) 值。

等价无穷小工具箱

公式 \(u \to 0\) 时的标准等价
\[\sin u \sim u,\quad 1-\cos u \sim \frac{u^2}{2},\quad \tan u \sim u,\quad \arctan u \sim u\] \[\ln(1+u) \sim u,\quad e^u-1 \sim u,\quad (1+u)^\alpha-1 \sim \alpha u\] \[x - \sin x \sim \frac{x^3}{6},\quad \arctan x = x - \frac{x^3}{3}+o(x^3)\]
例题：\(\int_0^{+\infty}\frac{\arctan(t^3)}{t^2}dt\)（TD Exercice 3.1）

双端积分 → 拆分：\(\int_0^1 + \int_1^{+\infty}\)。

瑕点 \(t\to 0^+\)：\(\arctan(t^3)\sim t^3\)，故 \(\frac{\arctan(t^3)}{t^2}\sim t\)，在 0 附近连续，无瑕点。✓

无穷远 \(t\to+\infty\)：\(\arctan(t^3)\to\pi/2\)（有界），故 \(\sim \frac{\pi/2}{t^2}\)，\(p=2>1\)，收敛。✓

结论：收敛。

例题：\(\int_0^1\frac{dt}{\tan(\sqrt{t})}\)（TD Exercice 3.4）

瑕点在 \(t=0\)。当 \(t\to 0^+\)：\(\sqrt{t}\to 0\)，\(\tan(\sqrt{t})\sim\sqrt{t}\)。

故 \(\frac{1}{\tan(\sqrt{t})} \sim \frac{1}{\sqrt{t}} = t^{-1/2}\)。

瑕积分 \(p = 1/2 < 1\)，收敛。

Dirichlet 与 Abel 判别法

公式 Dirichlet 判别法
\[\text{若}\;\left|\int_a^A f(x)\,dx\right| \le M\;(\text{积分有界}),\quad g(x)\;\text{单调}\to 0,\quad \text{则}\;\int_a^{+\infty}f(x)g(x)\,dx\;\text{收敛。}\]
公式 Abel 判别法
\[\text{若}\;\int_a^{+\infty}f(x)\,dx\;\text{收敛},\quad g(x)\;\text{单调有界},\quad \text{则}\;\int_a^{+\infty}f(x)g(x)\,dx\;\text{收敛。}\]
⚠ 关键区分 Dirichlet 要求「积分有界」而非「函数有界」

对于 \(\int \frac{\sin x}{x}dx\)，\(\sin x\) 本身有界（\(|\sin x|\le 1\)），但这不是 Dirichlet 所要求的条件！

你必须写：\(\left|\int_1^A \sin x\,dx\right| = |\cos 1 - \cos A| \le 2\)，即原函数有界。

考点 答题模板：Dirichlet 判别法

1. 「令 \(f(x) = \sin x\)（振荡部分），\(g(x) = 1/x\)（衰减部分）。」

2. 「\(\int_1^A f(x)\,dx = [-\cos x]_1^A = \cos 1 - \cos A\)，故 \(\left|\int_1^A \sin x\,dx\right| \le 2\)，积分有界。」

3. 「\(g(x) = 1/x\) 在 \([1,+\infty)\) 上单调递减，且 \(\lim_{x\to+\infty}g(x)=0\)。」

4. 「由 Dirichlet 判别法，\(\int_1^{+\infty}\frac{\sin x}{x}\,dx\) 收敛。」

例题：\(\int_1^{+\infty}\sin(2t)\arctan(4/t^3)\,dt\)（TD Exercice 7.1c）

令 \(f(t) = \sin(2t)\)，\(g(t) = \arctan(4/t^3)\)。

\(\int_1^A \sin(2t)\,dt = [-\frac{\cos 2t}{2}]_1^A\)，有界（\(\le 1\)）。

\(g(t) = \arctan(4/t^3)\) 单调递减趋于 0。

由 Dirichlet，\(\int\) 收敛。加绝对值后 \(|\sin(2t)|\cdot\arctan(4/t^3) \sim \frac{4|\sin(2t)|}{t^3}\)，由 \(\int\frac{dt}{t^3}\) 收敛知绝对收敛。

结论：绝对收敛。

绝对收敛与条件收敛

定义

绝对收敛：\(\int|f(x)|dx\) 收敛 → 自动保证 \(\int f(x)dx\) 也收敛。

条件收敛：\(\int f(x)dx\) 收敛但 \(\int|f(x)|dx\) 发散。依赖正负抵消。

考点 判断流程

① 先加绝对值，尝试直接放缩为 \(1/x^p\)（\(p>1\)）。若成功 → 绝对收敛。

② 若绝对值后发散，则用 Dirichlet/Abel 证明原积分收敛 → 条件收敛。

③ 若两者都做不到 → 可能发散（例如 \(\int\sin^2 x/x\)）。

经典判例速查
积分结论关键理由
\(\int_1^\infty\frac{\sin x}{x}dx\)条件收敛Dirichlet 证收敛；\(|\sin x|/x \ge \sin^2 x/x\) 发散
\(\int_1^\infty\frac{\cos x}{x^3}dx\)绝对收敛\(|\cos x|/x^3 \le 1/x^3\)，\(p=3>1\)
\(\int_1^\infty\sin(x^2)dx\)条件收敛Fresnel 积分，换元后 Dirichlet

混合型积分与含参讨论

方法 奇点隔离原则

对 \(\int_0^{+\infty}\) 型，必须从中间截断（通常在 \(x=1\)），分别处理两端。两端都收敛才整体收敛。

例题：\(\int_0^{+\infty}\frac{t^a}{1+t^2}dt\) 的收敛条件（TD Exercice 5.1）

拆为 \(\int_0^1 + \int_1^{+\infty}\)。

在 0 附近：\(1+t^2 \sim 1\)，被积函数 \(\sim t^a\)，瑕积分需 \(a > -1\)。

在 \(\infty\)：\(1+t^2 \sim t^2\)，被积函数 \(\sim t^{a-2}\)，无穷积分需 \(a-2 < -1\) 即 \(a < 1\)。

收敛条件：\(\boxed{-1 < a < 1}\)。

例题：\(\int_1^{+\infty}\frac{\ln t}{t^a}dt\) 的收敛条件（TD Exercice 5.5）

无穷积分。\(\ln t\) 增长极慢，对任意 \(\varepsilon > 0\)，\(\ln t \le t^\varepsilon\)（充分大时）。

所以 \(\frac{\ln t}{t^a} \le \frac{1}{t^{a-\varepsilon}}\)，需 \(a-\varepsilon > 1\)，即 \(a > 1\)。

当 \(a = 1\)：\(\int\frac{\ln t}{t}dt = \frac{(\ln t)^2}{2} \to \infty\)，发散。

收敛条件：\(\boxed{a > 1}\)。

例题：\(\int_0^1\frac{\ln t}{t^a}dt\) 的收敛条件（TD Exercice 5.6）

瑕积分（瑕点 0）。换元 \(t = 1/u\)，\(dt = -du/u^2\)：

\(\int_1^{+\infty}\frac{-\ln u}{u^{-a}}\cdot\frac{du}{u^2} = \int_1^{+\infty}\frac{\ln u}{u^{2-a}}du\)

由上一题，需 \(2-a > 1\) 即 \(a < 1\)。

收敛条件：\(\boxed{a < 1}\)。

拓展：变量代换降阶

拓展 高频振荡积分的处理

遇到 \(\int x^k\sin(x^m)dx\) 型，令 \(t = x^m\) 降低振荡频率。

\[\int_1^{+\infty}x^2\sin(x^4)\,dx \xrightarrow{t=x^4} \frac{1}{4}\int_1^{+\infty}\frac{\sin t}{t^{1/4}}dt\]

后者可用 Dirichlet 证收敛。

拓展 Frullani 积分（TD Exercice 13.2）
\[\int_0^{+\infty}\frac{f(at)-f(bt)}{t}\,dt = f(0)\ln\frac{b}{a}\]

条件：\(f\) 连续，\(\int_1^\infty f(t)/t\,dt\) 收敛。

应用：\(\int_0^{+\infty}\frac{e^{-at}-e^{-bt}}{t}dt = \ln(b/a)\)。

拓展 \(\int_0^{\pi/2}\ln(\sin t)\,dt = -\frac{\pi}{2}\ln 2\)（TD Exercice 8）

利用对称性 \(\int_0^{\pi/2}\ln(\sin t)dt = \int_0^{\pi/2}\ln(\cos t)dt = A\)，加上换元 \(t=2s\) 得到 \(I = 2A\)，最终 \(I = -\pi\ln 2\)。这类技巧在竞赛中常见。

拓展：理论深化

理论 极限为 0 不保证积分收敛

\(\lim_{t\to\infty}f(t)=0\) 是积分收敛的必要非充分条件（对正函数）。反例：\(f(t) = 1/t\)。

甚至存在 \(\lim f(t) \ne 0\) 但 \(\int f\) 收敛的函数（需要正负抵消）。

理论 收敛 + 导数收敛 → 极限为 0（TD Exercice 11.5）

若 \(f \in C^1\)，\(\int_0^\infty f\) 和 \(\int_0^\infty f'\) 都收敛，则 \(\lim_{t\to\infty}f(t)=0\)。

证明思路：\(f(t) = f(0)+\int_0^t f'(s)ds\)，两项极限都存在，故 \(f\) 的极限存在。若极限 \(\ne 0\) 则 \(\int f\) 发散（矛盾）。

理论 连续不趋于 0 但积分收敛（TD Exercice 11.4b）

构造：在每个整数 \(k\) 附近放一个高度为 1、宽度为 \(1/2^k\) 的「尖峰」，面积之和 \(= \sum 1/2^k = 1 < \infty\)。积分收敛但函数不趋于 0。
