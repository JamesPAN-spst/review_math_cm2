Quick Reference — 考点速查

⚠ 最高频易错点

  • p-判别法条件反转：无穷积分 p>1 收敛，瑕积分 p<1 收敛。口诀：远严近宽。
  • 极限比较法单向性：极限为 0 时只能从分母推分子（分母收敛→分子收敛），不能反推。
  • Dirichlet 条件：对振荡部分要求的是「积分有界」，而非函数本身有界。
  • 子空间并集：两个子空间的并集一般不是子空间，交集才是。
  • det(A+B) ≠ det A + det B：行列式对行/列线性，但不对矩阵加法线性。
  • det(λA) = λⁿ det A：不要忘记 n 次方。
