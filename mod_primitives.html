原函数与积分技巧

Primitives et techniques d'intégration

基本原函数表

公式 幂函数与指数
\[\int x^n\,dx = \frac{x^{n+1}}{n+1}+C \;(n \ne -1), \quad \int \frac{dx}{x} = \ln|x|+C\] \[\int e^x\,dx = e^x+C, \quad \int a^x\,dx = \frac{a^x}{\ln a}+C\]
公式 三角函数
\[\int \sin x\,dx = -\cos x+C, \quad \int \cos x\,dx = \sin x+C\] \[\int \frac{dx}{\cos^2 x} = \tan x+C, \quad \int \frac{dx}{\sin^2 x} = -\cot x+C\]
公式 反三角 / 根式
\[\int \frac{dx}{1+x^2} = \arctan x+C, \quad \int \frac{dx}{\sqrt{1-x^2}} = \arcsin x+C\] \[\int \frac{dx}{\sqrt{x^2+a}} = \ln\left|x+\sqrt{x^2+a}\right|+C \quad (a = \pm1)\]

最后一个公式来自课件 Exercice 1.1：对 \(g_a(t) = \ln(t+\sqrt{t^2+a})\) 求导可验证。

分部积分 (IPP)

公式 Integration by Parts
\[\int u\,dv = uv - \int v\,du\]

选取 \(u\) 的优先级口诀 LIATE：Log → Inverse trig → Algebraic → Trig → Exponential。越靠左越适合做 \(u\)（因为求导后更简单）。

考点 三种典型 IPP 场景

① 多项式 × 指数/三角：\(\int x^n e^x dx\)，令 \(u = x^n\)，IPP \(n\) 次直到多项式消失。

② 对数/反三角：\(\int \ln x\,dx\)，令 \(u = \ln x,\, dv = dx\)。同理 \(\int \arctan x\,dx\)。

③ 循环型：\(\int e^x \cos x\,dx\)，IPP 两次后出现原积分 \(I\)，联立 \(I = \cdots - I\) 解出。

例题：计算 \(\int t^2 e^{-t}\,dt\)（TD Exercice 4）

第一次：\(u=t^2,\, dv=e^{-t}dt \implies du=2t\,dt,\, v=-e^{-t}\)

\(= -t^2 e^{-t} + 2\int t e^{-t} dt\)

第二次：\(u=t,\, dv=e^{-t}dt \implies du=dt,\, v=-e^{-t}\)

\(= -t^2 e^{-t} + 2\left[-te^{-t}+\int e^{-t}dt\right] = -t^2 e^{-t}-2te^{-t}-2e^{-t}+C\)

\(= -(t^2+2t+2)e^{-t}+C\)

例题：计算 \(\int e^t \cos t\,dt\)（循环型 IPP）

令 \(I = \int e^t\cos t\,dt\)。

第一次：\(u=\cos t,\, dv=e^t dt\) → \(I = e^t\cos t + \int e^t\sin t\,dt\)

第二次：\(u=\sin t,\, dv=e^t dt\) → \(I = e^t\cos t + e^t\sin t - I\)

因此 \(2I = e^t(\cos t+\sin t)\)，即 \(I = \frac{e^t(\cos t+\sin t)}{2}+C\)。

换元法 (Changement de variable)

公式 基本形式
\[\int_a^b f(g(t))\,g'(t)\,dt = \int_{g(a)}^{g(b)} f(u)\,du, \quad u = g(t)\]
考点 三角代换一览

含 \(\sqrt{a^2-x^2}\)：令 \(x = a\sin\theta\)，利用 \(1-\sin^2=\cos^2\)。

含 \(\sqrt{a^2+x^2}\)：令 \(x = a\tan\theta\)，利用 \(1+\tan^2=\sec^2\)。

含 \(\sqrt{x^2-a^2}\)：令 \(x = a\sec\theta\) 或 \(x = a\cosh t\)。

例题：\(\int_0^{1/2}\sqrt{1-t^2}\,dt\)（TD Exercice 5.1）

令 \(t = \cos u\)，则 \(dt = -\sin u\,du\)。

当 \(t=0 \Rightarrow u=\pi/2\)，\(t=1/2 \Rightarrow u=\pi/3\)。

\(\int_{\pi/2}^{\pi/3} \sqrt{1-\cos^2 u}\cdot(-\sin u)\,du = \int_{\pi/3}^{\pi/2} \sin^2 u\,du\)

利用 \(\sin^2 u = \frac{1-\cos 2u}{2}\)：

\(= \left[\frac{u}{2}-\frac{\sin 2u}{4}\right]_{\pi/3}^{\pi/2} = \left(\frac{\pi}{4}-0\right)-\left(\frac{\pi}{6}-\frac{\sqrt{3}}{8}\right) = \frac{\pi}{12}+\frac{\sqrt{3}}{8}\)

考点 答题书写规范

必须写明：① 换元表达式 \(u = g(t)\)；② \(du = g'(t)\,dt\)；③ 新积分限（定积分时）。

漏写任意一项都可能被扣分。

三角函数积分

公式 降幂/线性化
\[\sin^2 x = \frac{1-\cos 2x}{2}, \quad \cos^2 x = \frac{1+\cos 2x}{2}\] \[\sin^3 x = \sin x(1-\cos^2 x), \quad \cos^3 x = \cos x(1-\sin^2 x)\]

奇数次幂：拆出一个 \(\sin\) 或 \(\cos\) 做换元。偶数次幂：用降幂公式线性化。

例题：\(\int_0^{\pi/2}\sin^3 t\,dt\) 与 \(\int_0^{\pi/2}\cos^4 t\,dt\)（TD Exercice 3）

\(\sin^3 t\)（奇次）：\(= \sin t(1-\cos^2 t)\)，令 \(u=\cos t\)：

\(\int_1^0 -(1-u^2)du = \int_0^1(1-u^2)du = 1-\frac{1}{3} = \frac{2}{3}\)

\(\cos^4 t\)（偶次）：\(\cos^4 t = \left(\frac{1+\cos 2t}{2}\right)^2 = \frac{1}{4}(1+2\cos 2t+\cos^2 2t)\)

其中 \(\cos^2 2t = \frac{1+\cos 4t}{2}\)，因此：

\(\cos^4 t = \frac{3}{8}+\frac{1}{2}\cos 2t+\frac{1}{8}\cos 4t\)

\(\int_0^{\pi/2} = \frac{3\pi}{16}+0+0 = \frac{3\pi}{16}\)

有理分式分解 (Éléments simples)

公式 分解规则

对 \(\frac{P(x)}{Q(x)}\)（\(\deg P \ge \deg Q\) 时先做多项式除法）：

\[\frac{1}{(x-a)(x-b)} = \frac{1}{a-b}\left(\frac{1}{x-a}-\frac{1}{x-b}\right)\]

不可约二次因子 \((x^2+px+q)\) 对应分子为 \(Ax+B\)（一次多项式）。

重根 \((x-a)^k\) 产生 \(k\) 个分式：\(\frac{A_1}{x-a}+\frac{A_2}{(x-a)^2}+\cdots+\frac{A_k}{(x-a)^k}\)。

考点 配方处理不可约二次

\(\int \frac{Ax+B}{x^2+px+q}dx\)：先凑分母导数 \(\frac{A}{2}(2x+p)\)，剩余部分配方后化为 \(\arctan\) 型。

\[\int \frac{dx}{x^2+x+\frac{5}{4}} = \int \frac{dx}{(x+\frac{1}{2})^2+1} = \arctan\!\left(x+\tfrac{1}{2}\right)+C\]
例题：\(\int \frac{t}{t^3-3t+2}dt\)（TD Exercice 7.1）

分母因式分解：\(t^3-3t+2 = (t-1)^2(t+2)\)。

设 \(\frac{t}{(t-1)^2(t+2)} = \frac{A}{t-1}+\frac{B}{(t-1)^2}+\frac{C}{t+2}\)

通分后比较系数（或代入特值 \(t=1, t=-2\) 速算）：

\(t=1 \Rightarrow 1=3B \Rightarrow B=\frac{1}{3}\)；\(t=-2 \Rightarrow -2=9C \Rightarrow C=-\frac{2}{9}\)。

比较 \(t^2\) 系数：\(0=A+C \Rightarrow A=\frac{2}{9}\)。

\(\int = \frac{2}{9}\ln|t-1| - \frac{1}{3(t-1)} - \frac{2}{9}\ln|t+2| + K\)

万能代换 (Weierstrass)

公式 半角代换
\[u = \tan\frac{t}{2} \implies \begin{cases} \sin t = \dfrac{2u}{1+u^2} \\[6pt] \cos t = \dfrac{1-u^2}{1+u^2} \\[6pt] dt = \dfrac{2\,du}{1+u^2} \end{cases}\]

适用场景：\(\int R(\sin t,\cos t)dt\)，当没有更高效的对称可利用时。

注意 万能代换不总是最优

如果被积函数关于 \(\sin\) 和 \(\cos\) 有特殊对称性（如只含 \(\cos 2t\)，或分子恰好是分母导数），先检查更简单的换元（如 \(u = \cos t\)、\(u = \sin t\)）再考虑万能代换。

例题：\(\int \frac{dt}{2\cos t+\sin t+1}\)（TD Exercice 12.2a）

令 \(u=\tan(t/2)\)：

\(\int \frac{1}{2\cdot\frac{1-u^2}{1+u^2}+\frac{2u}{1+u^2}+1}\cdot\frac{2}{1+u^2}du\)

分母通分：\(\frac{2(1-u^2)+2u+(1+u^2)}{1+u^2} = \frac{3+2u-u^2}{1+u^2}\)

所以 \(\int \frac{2\,du}{3+2u-u^2} = \int \frac{2\,du}{4-(u-1)^2}\)。

这是标准形式，用分式分解或令 \(u-1=2\sin\theta\) 处理即可。

指数型换元

考点 \(e^t\) 换元处理指数有理式

形如 \(\int R(e^t)dt\) 的积分，令 \(u = e^t\)，\(du = e^t dt = u\,dt\)，即 \(dt = du/u\)，化为有理分式。

\[\int \frac{e^t+1}{e^{2t}-5e^t+6}dt \xrightarrow{u=e^t} \int \frac{u+1}{u(u^2-5u+6)}du = \int \frac{u+1}{u(u-2)(u-3)}du\]

这类题在 TD Exercice 8 中大量出现。

拓展知识

拓展 双曲函数
\[\cosh t = \frac{e^t+e^{-t}}{2},\quad \sinh t = \frac{e^t-e^{-t}}{2},\quad \cosh^2 t - \sinh^2 t = 1\] \[\int \frac{dt}{\cosh^2 t} = \tanh t + C, \quad \int \frac{dt}{\sinh^2 t} = -\coth t + C\]

双曲万能代换：令 \(u = \tanh(t/2)\)，类似三角万能代换（TD Exercice 9）。

拓展 递推积分 (Intégrales de réduction)

典型递推关系（TD Exercice 10）：设 \(I_n = \int_0^{1/2}\frac{dt}{(t^2-1)^n}\)，通过 IPP 建立 \(I_{n+1}\) 与 \(I_n\) 的关系：

\[I_n = \frac{1}{2}\cdot\frac{(-4)^n}{3^n}+2n(I_n+I_{n+1})\]

另一经典递推：\(J_n = \int_0^{\pi/2}\sin^n x\,dx\)，有 \(J_n = \frac{n-1}{n}J_{n-2}\)，\(J_0 = \pi/2\)，\(J_1 = 1\)（Wallis 积分）。

拓展 Cauchy-Schwarz 积分不等式
\[\left(\int_a^b f(x)g(x)\,dx\right)^2 \le \int_a^b f(x)^2\,dx \cdot \int_a^b g(x)^2\,dx\]

等号成立当且仅当 \(f\) 与 \(g\) 成比例。应用：若 \(f(a)=0\)，\(f\in C^1\)，则 \(\int_a^b|f|^2 dx \le \frac{(b-a)^2}{2}\int_a^b|f'|^2 dx\)（TD Exercice 14）。

拓展 广义中值定理

设 \(g\) 连续，\(h \ge 0\) 连续，则存在 \(c \in [a,b]\) 使得：

\[\int_a^b g(t)h(t)\,dt = g(c)\int_a^b h(t)\,dt\]

TD Exercice 13.1 的内容。是积分中值定理的加权版本。
